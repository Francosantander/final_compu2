En la realizacion de este proyecto lo que se quiero hacer es un file server que además pueda validar coberturas geográficas.

En el mismo se podrán subir o descargar mapas geográficos en formato KML y a su vez el file server contará con la capacidad de validar si un punto en el plano(archivo kml), el cual se ingresarán como coordenadas en un formulario, se encuentra dentro del mapa geográfico que será pasado a su vez en el formulario mencionado anteriormente.

Elegí este proyecto ya que para empresas que venden, por ejemplo servicio de internet(Wireless o Fibra Óptica), les permitirá saber si un usuario al que le quieren vender internet, se encuentra dentro de su área de cobertura.

Para la elaboracion del mismo se usara:

- Aiohttp para levantar el server multicliente junto a la asyncio para realizar consultas asincronas
- Paralelismo mediante hilos para leer los mapas kml y verificar la cobertura
- Mecanismo IPC: Se utilizara memoria compartida
- Sincronismo: Se utilizara Locks para sincronizar los hilos entre sí
- Parseo de argumentos por línea de comandos(-p o --port y -s o --size)

Funcionalidades del server:

- Descargar archivos KML
- Subir archivos KML
- Verificar la cobertura de una persona en algun mapa seleccionado.
